# Cine-Shuffle v2 🎬

## New Features

- 🎲 Random movie/series fetch via TMDB API
- 📺 Embedded trailer in modal (no redirect)
- 🎬 Real posters & metadata
- 🔍 Genre + type filters
- 🌙 Dark mode toggle
- 💾 LocalStorage watchlist ready

## Setup

1. Replace `YOUR_TMDB_API_KEY` in:
   - `pages/api/random.js`
   - `pages/result.js`

2. Run:
```bash
npm install
npm run dev
```
